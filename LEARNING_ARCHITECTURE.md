# Türkçe Aşk — Образовательная архитектура

## Миссии вместо уроков. Путешествие вместо курса.

> Пользователь не «учит турецкий».
> Он проходит путь отношений и жизни в Турции — а язык осваивает по дороге.

---

## 0. Философия

| Обычное приложение | Türkçe Aşk |
|---|---|
| Уроки по темам | Миссии = реальные жизненные ситуации |
| «У Ахмета красная машина» | «Он написал `aşkım`, что ответить?» |
| Прогресс в % слов | Прогресс в «Готовности к жизни в Турции» |
| Грамматика отдельно | Грамматика внутри ситуации |
| Учишь ради экзамена | Учишься ради любви и реальной жизни |

Под игровой оболочкой — строгий каркас **CEFR (A1→A2→B1→B2)**. Пользователь видит миссии и карту, но система знает его официальный уровень.

## 1. Структура уровней CEFR

### A1 — «İlk Adımlar» (Первые шаги)
**Жизненный этап:** знакомство, флирт, первые свидания, выживание в городе.
**Языковая цель:** базовое выживание + первое общение в отношениях.
**Навыки на выходе:**
- Представиться, спросить имя/возраст/город
- Написать первое сообщение и ответить
- Сказать о чувствах простыми словами
- Заказать в кафе, поймать такси, купить базовое
- Настоящее время, личные местоимения, гармония гласных, отрицание, вопросы

### A2 — «Daha Yakın» (Ближе)
**Жизненный этап:** семья партнёра, быт, первые конфликты, традиции.
**Языковая цель:** уверенное бытовое общение, рассказ о прошлом.
**Навыки на выходе:**
- Общаться с его семьёй, вести себя за столом
- Рассказать о вчерашнем дне (прошедшее время)
- Обсудить ссору и помириться
- Объясниться у врача, в аптеке
- Падежи (направление, местонахождение), прошедшее время, притяжательность

### B1 — «Yeni Hayat» (Новая жизнь)
**Жизненный этап:** переезд, бюрократия, работа, подготовка к свадьбе.
**Языковая цель:** самостоятельность в Турции без переводчика.
**Навыки на выходе:**
- Снять квартиру, открыть счёт, оформить ВНЖ
- Пройти собеседование, общаться с коллегами
- Строить планы на будущее (будущее время, условия)
- Выражать мнение, аргументировать
- Будущее время, условное наклонение, придаточные

### B2 — «Tam Türk» (Совсем своя)
**Жизненный этап:** глубокие отношения, дети, карьера, идентичность.
**Языковая цель:** нюансы, юмор, эмоции, профессиональная речь.
**Навыки на выходе:**
- Понимать иронию, идиомы, культурный подтекст
- Поддержать в трудный момент, вести серьёзный разговор
- Общаться о беременности, детях, недвижимости
- Свободно спорить и убеждать
- Косвенная речь, причастия, деепричастия, аорист-нюансы

> **Архитектура заложена под C1/C2** — достаточно добавить новые ветки миссий и грамматические модули.

## 3. Анатомия миссии (9 шагов)

Каждая миссия — это полноценный мини-курс. Шаги идут последовательно, складываясь в одну историю.

```
🎯 Цель миссии     → «Он написал первым. Ответь так, чтобы захотелось продолжить»
📚 Новые слова     → 8–12 слов ситуации (карточки + озвучка + контекст)
⚡ Ключевые глаголы → 2–3 глагола с мини-спряжением в нужном времени
🧩 Грамматика      → 1 микро-тема, объяснённая через эту ситуацию
💬 Реальные фразы  → готовые конструкции «бери и пиши»
🎧 Аудирование     → послушай сообщение/диалог, пойми смысл
🎭 Диалог          → интерактивная переписка: выбираешь реплики, AI ведёт
🎮 Мини-игра       → собери фразу / вставь слово / подбери ответ
📝 Мини-тест       → 5–7 вопросов на всё из миссии
🔁 Повторение      → материал уходит в SRS-очередь
🏆 Награда         → XP, значок миссии, +«Готовность к жизни»
```

**Принцип:** грамматика никогда не подаётся «голой». В миссии «Первое сообщение» настоящее время изучается на фразах `seni düşünüyorum`, `ne yapıyorsun?` — а не на абстрактных таблицах.

## 4. Как связаны слова, глаголы, грамматика, диалоги

Внутри миссии всё крутится вокруг **одной коммуникативной задачи**.

**Пример: Миссия 3 «Первое свидание»**

| Слой | Содержание |
|---|---|
| **Слова** | buluşma, kafe, randevu, güzel, heyecanlı, birlikte, akşam, sonra |
| **Глаголы** | `buluşmak` (встречаться), `içmek` (пить), `gitmek` (идти) — в настоящем |
| **Грамматика** | Вопросы с `mı/mi`: «Kahve içmek ister misin?» |
| **Фразы** | «Bu akşam buluşalım mı?», «Seni eve bırakayım» |
| **Диалог** | Переписка-планирование свидания: ты выбираешь реплики |
| **Игра** | Собери «Benimle ___ ister misin?» из слов |
| **Тест** | Узнавание слов + правильный вопрос + ответ на сообщение |

Слова из миссии 3 переиспользуются в миссии 4 «Каждый день на связи» — так лексика закрепляется естественно.

## 5. AI-наставник «Elif Hoca»

Персона: тёплая турецкая учительница, которая «болеет» за твои отношения. Не робот-переводчик.

**Роли AI на протяжении миссии:**
1. **Объясняет** — «`-iyor` это как русское "-ю/-ешь", действие прямо сейчас»
2. **Подсказывает** — даёт намёк, не сразу ответ
3. **Проверяет свободный ответ** — пользователь пишет своё → AI оценивает и мягко правит
4. **Объясняет ошибку** — не «неверно», а «почти! ты сказал `seviyorsun`, но о себе — `seviyorum`»
5. **Культурный контекст** — «после еды у его мамы обязательно скажи `elinize sağlık`»
6. **Мотивирует** — «3 миссии до знакомства с его семьёй! 💪»
7. **Отвечает на любой вопрос** по грамматике/культуре в свободном чате

**Технически:** Claude API. Системный промпт задаёт персону Elif Hoca, уровень пользователя (A1–B2), текущую миссию и список его частых ошибок → ответы адаптивны.

## 6. Интервальное повторение (SRS) — для всего

Повторяется не только лексика, а **5 типов памяти**:

| Тип | Что повторяется | Формат повторения |
|---|---|---|
| Слова | выученные слова | карточка / квиз |
| Глаголы | формы спряжения | впиши форму |
| Грамматика | пройденные конструкции | собери фразу |
| Фразы | готовые выражения | ответь на ситуацию |
| **Ошибки** | где пользователь ошибался | целевой мини-дрилл |

Алгоритм Leitner (коробки 0→1→3→7→30 дней). Когда что-то «созревает» — на карте появляется значок 🔁 «Пора повторить», как отдельная мини-миссия.

## 7. Проверка знаний — экзамены уровня

В конце каждого уровня — **экзамен CEFR** из 6 секций:

1. **Лексика** — узнавание и перевод
2. **Грамматика** — выбор/построение форм
3. **Аудирование** — послушай → пойми
4. **Чтение** — короткий текст + вопросы
5. **Разговор** — ситуация → выбери/напиши ответ
6. **Письмо** — напиши сообщение по заданию (проверяет AI)

**Проходной балл — 70%.** Если секция провалена → AI составляет **персональный план повторения** именно слабых тем, и только после него открывает следующий уровень. Успех → значок уровня + обновление официального CEFR-статуса в профиле.

## 8. Мотивация и метрики прогресса

Пользователь видит не «% слов», а **жизненные метрики**:

- 🗺️ **Миссий пройдено:** 7 / 46
- 💬 **Диалогов завершено:** 12
- 🎯 **Ситуаций освоено:** 7 (знакомство, свидание, кафе…)
- 🇹🇷 **Готовность к жизни в Турции:** 18%  (растёт с каждой миссией)
- 🎓 **Официальный уровень:** A1 (в процессе)
- 🔥 Streak, XP, уровень-персона (Turist→Sultan), достижения — уже есть

«Готовность к жизни» — главный эмоциональный счётчик. Цель — довести до 100% = «Ты готова к жизни в Турции».

## 9. Игровые механики

- **Карта-путешествие** — извилистая тропа с узлами-миссиями (как Duolingo path / Candy Crush)
- **Замки** на будущих миссиях → предвкушение
- **3 звезды** за миссию (за тест, за диалог без ошибок, за идеальное произношение)
- **Сердечки-жизни** в тестах и играх
- **Сундуки** на узловых миссиях (после экзамена)
- **Парные реплики** в диалоге — «он» и «ты»
- **Озвучка двумя голосами** (мужской/женский)
- **Конфетти + звук** на завершении (уже есть)

```js
Mission = {
  id, level: 'A1', order: 3,
  emoji: '☕', title: 'Первое свидание',
  goal: 'Спланируй свидание и ответь на его сообщение',

  verbs: [verbIds...],
  grammarId,            // микро-тема
  phrases: [phraseIds...],
  dialog: [ {speaker:'he'|'you', tr, ru, options?} ... ],
  game: { type:'build'|'fill'|'match', data },
  test: [ {q, options, answer, explain} ... ],

```

Контент A1 на 80% уже существует (`data.js`, `phrases.js`, `verbs.js`, `gamification.js`) — миссии его **переиспользуют и связывают**, а не дублируют.

## 11. MVP vs будущие версии

### ✅ MVP (реализуемо сейчас, без новых API)
- **Карта-путешествие** (вертикальная тропа с узлами + замки)
- **Структура миссии** на существующем контенте: цель → слова → глаголы → грамматика → фразы → мини-игра → тест → награда
- **Первые 6–12 миссий A1**, собранные из текущих данных
- Прогресс: миссий пройдено, готовность к жизни, CEFR-статус в профиле
- Разблокировка по цепочке + 3 звезды
- SRS для слов/грамматики/ошибок (база уже есть)

### 🔜 v2.0 (нужен Claude API)
- **AI-наставник Elif Hoca** — объяснения, проверка свободных ответов, чат
- **Интерактивный диалог** с ветвлением, который ведёт AI
- Письменная секция экзамена с AI-проверкой

### 🔮 v3.0 (нужны медиа/нативные API)
- Озвучка носителями (муж/жен голоса)
- **Произношение** — запись голоса + оценка (Speech Recognition)
- **Аудирование** на студийных записях
- Полные экзамены A2/B1/B2
- Контент B1/B2 (миссии 25–46)
- Сертификат с проверкой уровня

## 12. Что меняется в текущем приложении

Сегодняшние разделы **не выбрасываются — переосмысляются**:

| Сейчас | Становится |
|---|---|
| Главная с сеткой тем | Карта-путешествие с миссиями |
| «Слова» (6 тем) | Банк лексики + источник для миссий |
| «Глаголы» | Модуль внутри миссий + справочник |
| «Грамматика» (5 уроков) | Микро-темы, встроенные в миссии |
| «Фразы» | Фразы внутри миссий + библиотека |
| «Переведи» | Инструмент + будущий вход к Elif Hoca |
| Квизы | Шаг «Тест» внутри миссии |

Ничего рабочего не теряется — всё становится **частями миссий**.
